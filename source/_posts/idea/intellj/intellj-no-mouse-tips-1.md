## 前言
这些操作前提是你没自己改过默认设置, 我都是在linux ubuntu 下使用, windows和macOS, 会略有不同. 这些东西就是用多了就熟练了, 没什么技巧

## 操作1: 查看最近操作的文件
查看最近操作的文件列表
```bash
Ctrl+E
```
查看最近操作的文件及内容
```bash
Ctrl+Shift+E
```

## 操作2: 声明方法之间跳转
```bash
Alt+Down 或 Up
```

## 操作3: 折叠方法体
```bash
Ctrl+Shift+Numpad 
```

## 操作4: 上下移动方法体或行
```bash
Ctrl+Shift+Down 或 Up
```

## 操作5: 在文件之间移动
```bash
 Ctrl+Right 或 Ctrl+Left
```

## 操作6: 打开终端
```bash
Alt+F12
```

## 操作7: 快速创建测试体
```bash
Alt+Insert
```
